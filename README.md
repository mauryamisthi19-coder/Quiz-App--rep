# Quiz Application

A responsive and interactive Quiz Application built using HTML, CSS, and JavaScript. This project allows users to answer multiple-choice questions, track their score, and view results instantly. The application is designed with a clean and user-friendly interface and works seamlessly across different devices.

## Features

- Multiple-choice quiz questions
- Animated interactive background
- Countdown timer for each question
- Dark/Light mode toggle
- Dynamic quiz functionality
- Score tracking system
- Modern responsive UI

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- dom manipulation
- Event Handling
- UI Design & Animations
